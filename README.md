# web-crawling

### axios를 이용한 간단한 웹 크롤링 예제

> 주요기능

* axios
* cherrio
* ajax 통신
* 동기화

> 사용법

자바스크립트 파일 실행

```bash
node index.js
```

1. axios로 해당 주소의 HTML DOM을 블러옵니다.
2. cheerio를 통해 DOM 에서 데이터를 추출합니다.

> 추후 수정 계획

1. 웹페이지로 만들기
2. 사이트 주소 입력 후 iframe으로 미리보기

    2-1. iframe에서 클릭한 문자를 자동으로 가져오도록 셋팅
3. 데이터 시각화
4. 데이터 텍스트 파일
